# TOP7 COFFEE - Menu

قائمة رقمية لمقهى TOP7 COFFEE تتضمن المشروبات الساخنة والباردة والشاي والحلويات.

## المميزات

- 🔍 بحث سهل عن المشروبات
- 📂 تصنيف المشروبات في فئات
- 💰 عرض الأسعار بشكل واضح
- 📱 تصميم متجاوب يعمل على جميع الأجهزة
- 💬 زر واتس مباشر للطلب

## التثبيت المحلي

```bash
npm install
npm start
```

## النشر على Vercel

1. ارفع المشروع إلى GitHub
2. اذهب إلى vercel.com
3. اختر import project واختر مستودعك
4. Vercel سينشر الموقع تلقائياً

## التحديث

لتحديث المشروبات والأسعار، عدّل الملف `App.js` وحدّث قائمة المشروبات.
